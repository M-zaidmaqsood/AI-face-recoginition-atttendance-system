AI Face Recognition Attendance System

An AI-based attendance system that uses face recognition to automatically mark attendance via webcam.
Built with Python, OpenCV, Streamlit, and an API-based backend.

=> Features

Face detection & recognition

Automatic attendance in CSV

Streamlit user interface

API-based architecture

Easy to train with custom datasets

// Tech Stack

Python

OpenCV

NumPy, Pandas

Streamlit

TensorFlow Lite

// Run the Project
# activate virtual environment
venv\Scripts\activate

# run backend API
python api/main.py

# run Streamlit UI
streamlit run ui/app.py
